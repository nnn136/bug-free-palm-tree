Программа «Weather Diary» (Дневник погоды)
Краткое описание программы
«Weather Diary» — это GUI‑приложение для учёта погодных наблюдений. Позволяет:

добавлять записи о погоде с указанием даты, температуры, описания и наличия осадков;

просматривать все записи в табличном виде;

фильтровать записи по дате и температуре;

сохранять данные в JSON‑файл и загружать их при следующем запуске;

использовать Git для версионирования данных.

Как запустить приложение
Требования:

Python 3.6+;

установленные библиотеки: tkinter, json, datetime, subprocess (для Git);

Git (опционально, для версионирования).

Порядок запуска:

Сохраните код в файл weather_diary.py.

Установите зависимости (если нужно):
pip install tkinter

Запустите приложение№
python weather_diary.py

При первом запуске будет создан файл weather_data.json
